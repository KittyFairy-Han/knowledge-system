# CSRF 跨站请求伪造

## 原理

|用| —— 登录 ——> |网|
|//| |站|
|//| <——下发 cookie—— |A|
|//|
|//| —— 访问 ——> |网|
|//| |站|
|户| <—— 引诱点击 —— |B|

## 防御措施

- token
- referer
- 隐藏令牌

# XSS 跨域脚本攻击

## 原理

- 不需要登录认证
- 向目标网站注入脚本即 script

## 防御措施

- 让插入的脚本不可执行
