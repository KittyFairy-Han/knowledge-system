# webpack 

## 为什么要打包和构建

#### 代码层面

- 体积更小-->加载更快：Tree-Shaking（去除冗余）、压缩、合并

- 编译高级的语言和语法：TS、ES6+、模块化、less、scss

- 兼容性和错误检查：Polyfill、postcss、eslint

#### 流程方面

- 更加统一和高效的 开发环境

- 统一的 构建流程和产出标准

- 更便捷的 集成公司的构建规范（提测、上线）
