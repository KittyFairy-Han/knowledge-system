Greensock动画平台 (GSAP)是一个出色的 Web 动画库，因为它使您能够对任何可通过 JavaScript 访问的内容进行动画处理，包括 DOM 元素、SVG、通用对象、画布等。
GSAP 是一个很棒的工具，可以用来构建简单到非常复杂的基于物理的动画。它允许开发人员对运动进行排序并动态控制动画。
