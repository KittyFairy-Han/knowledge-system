# link rel

rel 表示“关系 (relationship) ”,属性值表示项的链接方式与包含它的文档之间的关系。

<!-- css样式 -->
<link rel="stylesheet" href="main.css" />
<!--可以在收藏夹中显示出图标-->
<link rel="Bookmark" href="images/tab.png" />
<!--可以在地址栏中显示出图标-->
<link rel="icon" href="favicon.ico" />
<!--  -->
<link rel="preload" as="script" href="main.js" />
